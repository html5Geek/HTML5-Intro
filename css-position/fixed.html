Key points need to be noted for Fixed position:

  1. "Fixed position" can be used when u need to completely remove an element from the structure flow.
  2. same like absolute position, neighbouring elements will be effected with fixed position.
  3. An absolute positioned element can be styled corresponding to the "Browser View Port".
