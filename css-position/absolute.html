Key points need to be noted for Absolute position:

  1. "Absolute position" can be used when u need to completely remove an element from the structure flow.
  2. With absolute position, neighbouring elements will be effected since one element is removed in the flow.
  3. An absolute positioned element can be styled corresponding to the "Browser window".
